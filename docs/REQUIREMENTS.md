# krav

## det du trenger

- Node.js 18+ (anbefalt: nyeste LTS)
- npm 9+

## env-variabler

Lag en `.env`-fil i rotmappa med:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

Eksempel:

```env
VITE_SUPABASE_URL=https://your-project-id.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

## supabase-krav

- Et Supabase-prosjekt
- Auth aktivert (epost/passord)
- Kommentar-tabell/policy satt opp så det matcher appen

## nettleser

- Moderne nettleser med støtte for ES-moduler og `fetch`
- Støtte for autoplay/mutet video for best opplevelse i hero-seksjonen
